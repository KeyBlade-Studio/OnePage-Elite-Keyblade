Elite OnePage Template
-----------------------
A clean, modern, and responsive template for multipurpose use.

Features:
- Fully responsive and mobile-friendly
- Easy color customization
- Optimized for speed and SEO
- Well-structured and clean code

Installation:
1. Download and extract the zip file.
2. Open "index.html" in your browser to preview.
3. Customize colors in "style.css" (instructions in "/Documentation/").

Customization Guide:
- To change primary and accent colors, edit the CSS variables at the top of "style.css".
- To add custom sections, modify "index.html" and follow the existing structure.
