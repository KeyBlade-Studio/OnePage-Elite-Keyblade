Elite OnePage Template - Documentation

Thank you for purchasing Elite OnePage Template. This documentation will help you install, customize, and use the template.

📌 1. Files & Structure

The template includes the following files and folders:

Elite-OnePage/
│── 📁 assets/
│   │── 📁 css/           → Stylesheets (style.css)
│   │── 📁 favicons/      → Favicon images
│   │── 📁 images/        → Logos & other images
│   │── 📁 js/            → JavaScript files (script.js)
│   │── 📁 vendor/        → Third-party libraries (AOS, Bootstrap)
│── 📁 documentation/     → This documentation
│── index.html            → Main HTML file

⚙️ 2. Installation

To use this template, simply extract the ZIP file and open index.html in a browser.

If you are uploading to a server, place all files in your server's root folder.

🎨 3. Customization

You can change the main colors by editing assets/css/style.css. Example:

:root {
    --primary-color: #ff007f;
    --accent-color: #00faff;
}

💡 4. FAQ & Troubleshooting

Q: The template is not displaying correctly?

A: Make sure you are using a modern browser (Chrome, Firefox, Edge).
